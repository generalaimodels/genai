# Markdown Documentation Converter Backend

SOTA Markdown to Documentation Website Converter API with support for:
- Markdown (.md)
- Jupyter Notebooks (.ipynb)
- MDX (React Markdown with JSX)
- R Documentation (.Rd/.Rdx)
- reStructuredText (.rst)

## Features

- Fast async document scanning and parsing
- Full-text search with inverted index
- RESTful API for frontend integration
- Multiple documentation format support
- Premium HTML rendering with syntax highlighting
- Math equation rendering (KaTeX)
- Mermaid diagram support

## Installation

```bash
pip install -e .
```

## Running

```bash
python main.py
```

Server starts on http://localhost:8000
